Password Checker (Java)

This is a simple Java project I built to evaluate the strength of a password using basic validation rules.

What it does
- Checks for the presence of uppercase and lowercase letters  
- Verifies if the password contains numeric digits  
- Provides a basic assessment of password strength  

Built with
- Java

How to run
1. Compile the files:
   javac *.java

2. Run the program:
   java PasswordChecker

Why I made this
I created this project while learning Java to practice core programming concepts like conditional logic, input validation, and working with multiple classes.
